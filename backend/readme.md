# ZAVIA Backend
## Backend Routes
**Backend url** 
http://localhost:8080

**User Routes**
- Register - /user/register
- Login - /user/login
- Mentor request - /user/mentor/request

**Admin Routes**

- Mentor Approval - /admin/mentor/{approval status}

**Mentor Routes**

- Get all mentors - /mentor

**Session Routes**

- Get session - /session/
- Book session - /session/create
- Cancel session - /session/cancel
